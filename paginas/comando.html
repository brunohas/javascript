Comandos do console

  • console.log() → imprime mensagem no console
  • console.info() → exibe informações informativas no console
  • console.warn() → registra mensagens de alerta no console
  • console.error() → registra mensagens de erro no console
  • console.debug() → emitir uma mensagem de depuração no console
  • console.trace() → exibir a sequência de chamadas de função que levaram o usuário à posição atual no código
  • console.table() → exibir dados em formato de tabela no console
  • console.dir() → exibe uma lista interativa das propriedades de um objeto JavaScript no console
  • console.group() → agrupa mensagens no console, facilitando a visualização de saídas relacionadas no console
  • console.groupEnd() → encerra a agrupação de mensagens no console
  • console.groupCollapsed() → organiza os registros do console em grupos ocultos
  • console.count() → conta o número de vezes que um comando foi executado
  • console.countReset() → reseta a contagem para um rótulo específico
  • console.time() → registra o tempo de execução de um comando
  • console.clear() → limpa o console

Comandos do alert

  • alert() → exibe uma caixa de diálogo com uma mensagem
  • confirm() → exibe uma caixa de diálogo com uma mensagem e dois botões: "OK" e "Cancelar"
  • prompt() → exibe uma caixa de diálogo com uma mensagem e um campo de entrada para o usuário digitar um valor

Comandos para selecionar elementos

  • document.getElementsByTagName(' ') → seleciona o(s) elemento(s) pelo nome da tag.

    • exemplo: .getElementsByTagName('p'), seleciona todas as tags <p>, se quiser escolher somente uma dessas tags <p> colocar o número equivalente, ex: [0] p/ primeira tag <p>, [1] p/ segunda tag <p> e assim adiante...

  • document.getElementById(' ') → seleciona o elemento pelo id.

    • exemplo: .getElementById('inum'), seleciona o elemento que tem o id="inum".

  • document.getElementsByName(' ') → seleciona o(s) elemento(s) pelo nome da propriedade name=" ".

    exemplo: .getElementsByName('num1'), seleciona o(s) elemento(s) que tem a propriedade name="num1".

  • document.getElementsByClassName(' ') → seleciona o(s) elemento(s) pelo class.

    • exemplo: .getElementsByClassName('msg'), seleciona o(s) elemento(s) que tem o class='msg'.

  • document.querySelector(' ') → seleciona o elemento específico.

    • exemplo: .querySelector('div.img'), seleciona a tag <div> com o class="img".

  • document.querySelectorAll(' ') → seleciona o(s) elemento(s) específico(s).

    • exemplo: .querySelectorAll('.msg'), seleciona todos os elementos que tem o class="msg".

Comandos para manipular elementos

  • elemento.innerText(' ') → atribui texto ao elemento selecionado, sem incluir tags HTML ou estilos CSS.
  • elemento.innerHTML(' ') → atribui texto ao elemento selecionado, incluindo tags HTML e estilos CSS.
  • document.TAG_HTML.style.CSS = ' ' → altera o estilo CSS de uma tag HTML.

    • exemplo: document.div.style.background = '#000', muda cor de fundo da tag <div>.

  • document.write(' ') → escreve na tela.

    exemplo: document.write('Bem-Vindo!'), escreve "Bem-Vindo!" na tela.

  • document.createElement(' ') → cria um elemento HTML.

    • exemplo: document.createElement('img'), cria um elemento <img>.

  • document.removeChild(' ') → remove um elemento filho.

    • exemplo: document.removeChild(img), remove a tag <img>.

  • document.appendChild(' ') → adiciona um elemento filho.

    • exemplo: div.appendChild(img), adiciona a tag <img> dentro da tag <div>.

  • document.replaceChild(' ') → substitui um elemento filho.

    • exemplo: div.replaceChild(h1, h2), substitui as tags <h2> por <h1> que estão dentro da tag <div>.

  • document.setAttribute(' ') → cria ou alterar um atributo para a tag escolhida.

    • exemplo: img.setAttribute('id', 'foto'), cria um id="foto" para a tag <img>.

  • elemento.classList.add(' ') → adiciona uma ou mais classes ao elemento selecionado.

    • exemplo: img.classList.add('foto', 'dest'), adiciona a classe "foto" e "dest" ao elemento <img>.

  • elemento.classList.toggle(' ') → adiciona ou remove uma classe ao elemento selecionado.

    • exemplo: img.classList.toggle('dest'), adiciona a classe "dest" ao elemento <img> se ela não estiver adicionada, ou remove se ela estiver adicionada.

  • elemento.classList.contains(' ') → verifica se um elemento possui uma classe.

    • exemplo: img.classList.contains('foto'), verifica se a classe "foto" está no elemento <img>.

  • elemento.classList.replace(' ', ' ') → substitui uma classe existente por outra.

    • exemplo: img.classList.replace('foto', 'dest'), substitui a classe "foto" por "dest" no elemento <img>.

  • elemento.classList.remove(' ') → remove um valor de classe.

    • exemplo: img.classList.remove('foto'), remove a classe "foto" do elemento <img>.

Comandos para Strings

  • .charAt( ) → retorna o caractere na posição especificada de uma string.
  • .charCodeAt( ) → retorna um número que representa o valor de um caractere em uma string, de acordo com a tabela Unicode.
  • .concat( ) → junta as strings.
  • .endsWith( ) → verifica se a string termina com o valor especificado, retornado um valor booleano.
  • .includes( ) → verifica se a string contém um valor especificado, retornado um valor booleano.
  • .indexOf( ) → retorna o índice do caractere especificado.
  • .lastIndexOf( ) → retorna o índice da última ocorrência do valor especificado.
  • .length( ) → retorna o comprimento de uma string.
  • .localeCompare( ) → compara duas strings e retorna um número que indica a ordem alfabética entre elas.
  • .match( ) → retorna um array com os trechos do texto que passaram na verificação.
  • .repeat( ) → repete a string.
  • .replace( , ) → substitui uma parte de um string por uma substring.
  • .search( ) → retorna o índice da primeira correspondência entre uma expressão regular e uma string, se não houver retorna -1.
  • .slice( ) → extrai uma parte da string a partir de um ponto inicial e um ponto final.
  • .split( ) → divide uma string em um array de substrings e retorna um array.
  • .startsWith( ) → verifica se a string começa com o valor especificado, retornando uma valor booleano.
  • .substr( ) → extrai caracteres de uma string a partir de um ponto inicial e um ponto final.
  • .substring( ) → extrai uma parte de uma string, entre dois índices específicados.
  • .toLocaleUpperCase( ) → converte uma string para maiúsculas de acordo com as regras da localidade.
  • .toLocaleLoweCase( )→ converte uma string para minúsculas de acordo com as regras da localidade.
  • .toLowerCase( ) → converte os caracteres tudo para minúscula.
  • .toUpperCase( ) → converte os caracteres tudo para maiúscula.
  • .trim( ) → remove espaços em branco no início e no fim da string.
  • .trimLeft( ) → remove espaços em branco no início da string.
  • .trimRigth( ) → remove espaços em branco no final da string.

  • .isInteger( ) → verifica se o valor é número inteiro.
  • .isNan( ) → verifica se o valor é Nan (Not A Number) ou não.
  • .isSafeInteger( ) → verifica se o valor é um número seguro.
  • Math.ceil() → arredonda para cima
  • Math.floor() → arredonda para baixo
  • .parseFloat( ) → converte em número real.
  • .parseInt( ) → converte em número inteiro.
  • .replace(' ', ' ') → troca caracteres
  • .toExponential( ) → formata números em notação exponencial.
  • .toFixed( ) → formata um número com um número específico de casas decimais.
  • .toPrecision( ) → formata um número para uma precisão específica. Ele é uma função incorporada ao objeto Number.
  • .toString → converte para string.
  • .valueOf( ) → converte em um valor primitivo.
  • new Date() → mostra dia, mês, ano, data e hora atual
  • new Date() + .getHours → mostra o horário atual
  • new Date() + .getMinutes → mostra o minuto atual
  • new Date() + .getSeconds → mostra o segundo atual
  • new Date() + .getMilliseconds → mostra o milissegundo atual
  • new Date() + .getDay → mostra o dia da semana atual
  • new Date() + .getDate → mostra o dia do mês atual
  • new Date() + .getMonth → mostra o mês atual
  • new Date() + .getFullYear → mostra o ano atual
  • .toLocalString( ) → converte um número para um string, já arredondando e convertendo para a moeda do país (decimal - representar número simples / currency - representar em valor monetário / percent - representar formato percentual)

    • .toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'}) → formata o número em valor monetário (.toLocaleString = string localizada / 'pt-BR' = localiza em português / style: currency = mostrar em valor monetário / currency: 'BRL' = real brasileiro)

Comandos para Arrays

  • .at( ) → acessa elementos de um array usando um índice, que pode ser positivo ou negativo.
  • .concat( ) → mescla 2 ou mais arrays.
  • .copyWithin( , , ) → copia uma sequência de elementos de um array para outra localização no mesmo array.
  • .entries( ) → retorna um objeto iterador que permite percorrer os pares índice/elemento de um array.
  • .every( ) → verifica se todos os elementos de um array atendem a uma condição específica, retornando true ou false.
  • .fill(elemento, início, fim) → preenche um array com um valor específico, a partir de um índice inicial até um índice final.
  • .filter( ) → cria um novo array com todos os elementos que passaram no teste implementado pela função fornecida.
  • .find( ) → retorna o valor do primeiro elemento do array que satisfizer a função de teste provida. Caso contrario, undefined é retornado.
  • .findIndex( ) → retorna o índice no array do primeiro elemento que satisfizer a função de teste provida. Caso contrário, retorna -1, indicando que nenhum elemento passou no teste.
  • .findLast( ) → retorna o último elemento do array que satisfizer a função de teste provida. Caso contrário, undefined é retornado.
  • .findLastIndex( ) → pesquisa o array de trás para frente e retorna o índice do primeiro elemento encontrado, ou -1 se o elemento não estiver presente.
  • .flat( ) → transforma um array em um novo array, concatenando os subarrays em um único array (.flat(Infinity) concatena todos os subarrays).
  • .flatMap( ) → mapeia cada elemento de um array e nivela o resultado em um novo array.
  • .forEach( ) → executa uma função fornecida uma vez para cada elemento do array.
  • .from( ) → cria um novo array a partir de um array-like ou de um iterable object.
  • .fromAsync( ) → cria uma nova instância de um array copiada superficialmente a partir de um objeto async iterable, iterable ou array-like.
  • .includes( ) → verifica se um array contém um determinado elemento, retornando true ou false.
  • .indexOf( ) → busca valor dentro do array e retorna o índice da primeira ocorrência do valor especificado, ou -1 se o valor não for encontrado.
  • .isArray( ) → verifica se o valor passado é um array.
  • .join( ) → junta todos os elementos de um array em uma string.
  • .keys( ) → serve para criar um array com as chaves de um objeto.
  • .lastIndexOf( ) → busca valor dentro do array e retorna o índice da última ocorrência do valor especificado, ou -1 se o valor não for encontrado.
  • .length( ) → quantidade de elementos dentro do array.
  • .map( ) → cria um novo array a partir de um array existente. Ele aplica uma função a cada elemento do array original.
  • .of( ) → cria um novo array com os elementos passados entre parênteses.
  • .pop( ) → remove o último elemento de um array e o retorna, alterando o comprimento do array.
  • .push( ) → adiciona um ou mais elementos ao final de um array e retorna o novo comprimento do array.
  • .reduce( ) → reduz um array a um único valor.
  • .reduceRight( ) → reduz um array em um único valor, da direita para a esquerda.
  • .reverse( ) → inverte a ordem dos elementos de um array.
  • .shift( ) → remove o primeiro elemento do array e o retorna, alterando o comprimento do array.
  • .slice( , ) → copia uma parte de um array, ou seja, para criar um subarray. O array original não é alterado.
  • .some( ) → verifica se pelo menos um elemento de um array atende a uma determinada condição, retornado true ou false (não modifica o array).
  • .sort( ) → ordena os elementos de um array em ordem crescente ou alfabética.
  • .splice( ) → adiciona elementos em uma posição específica do array.
  • .toLocaleString( , ) → converte os elementos de um array em uma string.
  • .toReversed( ) → inverte a ordem dos elementos de um array.
  • .toSorted( ) → ordena os elementos de um array em ordem crescente, retornando um novo array. O array original não é alterado.
  • .toSpliced( , ) → exclui, adiciona e substitui elementos em um array. Ele também cria um novo array de forma mais eficiente do que os métodos slice( ) e concat( ).
  • .toString( ) → converter um array em uma string.
  • .unshift( ) → adiciona um ou mais elementos no início do array.
  • .values( ) → retorna um array com os valores das propriedades enumeradas de um objeto.
  • .with( , ) → altera um elemento com base no índice indicado, retornando um array com os valores indicados.
  • [Symbol.iterator]( ) → retorna um objeto iterador de array que produz o valor de cada índice no array.

Comandos para Object

  • .assign( ) → copia os valores de um ou mais objetos para um objeto destino, retornando o objeto de destino modificado.
  • .create( ) → cria um novo objeto usando um objeto existente como protótipo do objeto recém-criado.
  • .defineProperties( ) → defini ou modifica as propriedades de um objeto, retornando o objeto modificado.
  • .defineProperty( ) → adiciona ou modifica propriedades em objetos.
  • .entries( ) → retorna um array de pares de elemento / índice de um objeto.
  • .freeze( ) → congela um objeto, não deixa ele ser modificado se tornado um objeto imutável.
  • .fromEntries( ) → transforma uma lista de pares elemento / índice em um objeto.
  • .getOwnPropertyDescriptor( ) → permite obter um descritor de propriedades para um objeto.
  • .getOwnPropertyDescriptors( ) → retorna uma lista de todos os descritores de propriedade de um objeto.
  • .getOwnPropertyNames( ) → retorna um array com os nomes de todas as propriedades de um objeto.
  • .getOwnPropertySymbols( ) → retorna um array com todas as propriedades de símbolo de um objeto.
  • .getPrototypeOf( ) → obter o protótipo de um objeto. O protótipo é o valor da propriedade interna [[Prototype]].
  • .groupBy( ) → agrupa os elementos de um iterável fornecido de acordo com os valores de string retornados por uma função de retorno de chamada fornecida. O objeto retornado tem propriedades separadas para cada grupo, contendo arrays com os elementos no grupo.
  • .hasOwn( ) → retorna true se o objeto especificado tiver a propriedade indicada como sua própria propriedade. Se a propriedade for herdada, ou não existir, o método retorna false.
  • .hasOwnProperty( ) → verifica se um objeto possui uma propriedade específica como sua própria. Ele retorna um valor booleano, true ou false, indicando se a propriedade existe.
  • .is( ) → verifica se dois valores são iguais.
  • .isExtensible( ) → verifica se um objeto pode ter novas propriedades adicionadas a ele.
  • .isFrozen( ) → verifica se um objeto está congelado.
  • .isPrototypeOf( ) → verifica se um objeto é herdeiro de outro.
  • .isSealed( ) → verifica se um objeto está selado.
  • .keys( ) → extrai as chaves de um objeto, retornando um array com as strings correspondentes a cada propriedade.
  • .preventExtensions( ) → impede que novas propriedades sejam adicionadas a um objeto. É usado para garantir que a estrutura de um objeto permaneça consistente.
  • .propertyIsEnumerable( ) → verifica se uma propriedade é enumerável, retornando um valor booleano.
  • .seal( ) → sela um objeto, impedindo a adição de novas propriedades e marcando as existentes como não configuráveis.
  • .setPrototypeOf( ) → configura o protótipo de um objeto para outro objeto ou null.
  • .toLocaleString( ) → retorna uma string representando este objeto.
  • .toString( ) → retorna uma string que representa um objeto.
  • .values( ) → retorna um array de valores de propriedade enumeráveis com chave de string do próprio objeto.
  • .valueOf( ) → converte um objeto em um valor primitivo.
