Eventos DOM

Os eventos DOM em JavaScript são ações que ocorrem no navegador e que podem ser capturadas e manipuladas pelo JavaScript. A seguir vou apresentar algumas delas, ou você pode clicar aqui para ir na página da W3C Schools com todos os eventos.

TouchEvent

Manipula eventos que ocorrem quando o usuário toca na tela de um dispositivo touchscreen.

  • ontouchstart=" " → quando o dedo é colocado em uma tela sensível ao toque.
  • ontouchend=" " → quando o dedo é removido de uma tela sensível ao toque.
  • ontouchcancel=" " → quando o toque é interrompido.
  • ontouchmove=" " → quando o dedo é arrastado pela tela.

MouseEvent

Manipula eventos que ocorrem quando o mouse interage com o documento HTML.

  • onclick=" " → quando clicar no elemento.
  • oncontextmenu=" " → quando clicar no elemento com mouse direito.
  • ondblclick=" " → quando clicar 2 vezes no elemento.
  • onmouseenter=" " → quando o cursor do mouse 'entrar' no elemento.
  • onmouseout=" " → quando o cursor do mouse 'sair' de qualquer elemento filho, bem como do elemento selecionado.
  • onmouseleave=" " → quando o cursor do mouse 'sair' do elemento selecionado.
  • onmousemove=" " → quando o cursor do mouse se move sobre o elemento.
  • onmouseover=" " → quando o cursor do mouse passa sobre o elemento, dispara a function.
  • onmousedown=" " → quando o mouse e pressionado sobre o elemento.
  • onmouseup=" " → quando o mouse não estiver mais pressionado sobre o elemento.

UiEvent

Manipula eventos que são acionados pela interface do usuário.

  • onabort=" " → quando o carregamento de um arquivo é abortado (evento ocorre quando um download é abortado, não quando ocorre um erro).
  • onbeforeunload=" " → quando um documento está prestes a ser descarregado (este evento permite que você exiba uma mensagem de confirmação para informar ao usuário se ele deseja permanecer ou sair da página).
  • onerror=" " → quando ocorrer um erro ao carregar um arquivo externo (ex: um documento, uma imagem, um script, etc).
  • onload=" " → quando um objeto termina de carregar (é usado com mais frequência dentro do elemento <body> para executar um script após a página ter sido completamente carregada).
  • onresize=" " → quando a janela do navegador é redimensionada.
  • onscroll=" " → quando a barra de rolagem de uma elemento está sendo rolada.
  • onselect=" " → ocorre depois que algum texto é selecionado em um elemento (é usado principalmente em elementos <input:text> ou <textarea>).
  • onunload=" " → quando o usuário sai da página ou a janela do navegador é fechada (também é acionado quando o usuário recarrega a página).

DragEvent

Manipula eventos que ocorrem quando elementos são arrastados ou soltos.

  • ondrag=" " → quando o elemento estiver sendo arrastado.
  • ondragend=" " → quando o elemento terminou de ser arrastado.
  • ondragenter=" " → quando o elemento arrastado entra no destino de soltar.
  • ondragleave=" " → quando o elemento arrastado sai do destino de soltar.
  • ondragover=" " → quando o elemento arrastado está sobre o destino de soltar.
  • ondragstart=" " → quando o usuário começa a arrastar o elemento.
  • ondrop=" " → quando o elemento arrastado é solto no alvo.

KeyboardEvent

Manipula eventos que ocorrem quando um usuário pressiona uma tecla no teclado.

  • onkeydown=" " → quando o usuário pressiona uma tecla (usado p/ detectar quando teclas modificadoras são pressionadas).
  • onkeypress=" " → quando o usuário pressiona e solta uma tecla (não é disparado em teclas que não geram caracteres, ex: F1~F12 / tab / esc / etc).
  • onkeyup=" " → quando o usuário solta uma tecla (usado p/ detectar quando teclas modificadoras são liberadas).

AnimationEvent

Manipula eventos que ocorrem quando uma animação CSS é executada.

  • animationstart=" " → quando uma animação CSS for iniciada.
  • animationend=" " → quando uma animação CSS for concluída.
  • animationiteration=" " → quando uma animação CSS for repetida.

ClipboardEvent

Manipula eventos que ocorrem quando a área de transferência é modificada.

  • oncopy=" " → quando um usuário copia o conteúdo de um elemento.
  • oncut=" " → quando um usuário corta o conteúdo de um elemento.
  • onpaste=" " → quando um usuário cola o conteúdo do elemento copiado.

FocusEvent

Manipula eventos que ocorrem quando elementos recebem ou perdem o foco.

  • onblur=" " → quando o elemento perde o foco.
  • onfocus=" " → quando o elemento ganha o foco.
  • onfocusin=" " → quando o elemento está prestes a ganhar foco.
  • onfocusout=" " → quando o elemento está prestes a perder foco.

Exemplo: div#menu.addEventListener('click', nome função)

Propriedades de eventos

As propriedades fornecem detalhes sobre o evento que foi disparado, essas propriedades permitem que o manipulador de eventos seja mais inteligente e útil. Clique aqui para ir na página da W3schools que tem todos os tipos de propriedades dos eventos DOM.

Métodos de eventos

Os métodos de eventos em JavaScript permitem que o navegador capture ações do usuário, como cliques, pressionamentos de teclas e carregamentos de páginas. Essas ações podem ser manipuladas para tornar a página web mais interativa. Segue os métodos de eventos HTML:

  • .getModifierState( ) → retorna true se a tecla modoficadora especificada for pressionada ou ativada (MouseEvent).
  • .getTargetRanges( ) → retorna um array contendo intervalos de destino que serão afetados pela inserção/exclusão (InputEvent).
  • .preventDefault( ) → cancela o evento se ele for cancelável, o que significa que a ação padrão que pertence ao evento não ocorrerá (Event).
  • .stopImmediatePropagation( ) → impede que outros ouvintes do mesmo evento sejam chamados (Event).
  • .stopPropagation( ) → impede a propagação de um evento durante o fluxo do evento (Event).
