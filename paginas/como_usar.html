Como usar o JavaScript?

Para utilizar o JavaScript em conjunto com o HTML e o CSS, utilizamos a tag <script> e </script>, mais o parâmetro function (+ um nome para a função) para incorporar código, todos os comandos em JavaScript devem ser colocado entre essas tags. Uma ferramenta muito útil para explorar o JavaScript em sites diversos, é o DevTools, para abrir ele basta clicar com o botão direito do mouse na página desejada e clicar em Inspecionar, assim abrindo uma nova janela com o código do site. Para fazer edição do site, colocamos os códigos no Console, e para descobrir os nomes dados aos parâmetros basta clicar na setinha no canto superior direito e passar por cima do item que você deseja saber o nome.

  • window.alert(' janela com alerta na tela ')
  • window.confirm(' janela p/ ok ou cancel ')
  • window.prompt(' janela p/ pergunta ao usuário ') retorna em string
  • document.write(' escreve na tela ') pode usar tags HTML dentro
  • .innerText → retorna o conteúdo de texto de um elemento HTML, sem incluir tags HTML ou estilos CSS.
  • .innerHTML → retorna o conteúdo de texto de um elemento HTML, incluindo suas tags HTML e estilos CSS.
