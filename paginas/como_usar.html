Como usar o JavaScript?

Para utilizar o JavaScript em conjunto com o HTML e o CSS, utilizamos a tag <script> e </script> para incorporar código. Todos os comandos em JavaScript devem ser colocado entre essas tags. Por exemplo: 

                    
    <script>
        
    </script>
