O que é JavaScript?

JavaScript é uma linguagem de programação que permite a criação de conteúdos dinâmicos, controle de mídias e animações para deixar seu site mais interativo e interessante. Ele é uma das principais tecnologias da web, juntamente com o HTML e CSS.

O que faz JavaScript?

A linguagem de programação JavaScript permite ao desenvolvedor implementar diversos itens de alto nível de complexidade em páginas web como animações, mapas, menu suspenso, formulários, gráficos animados, informações que se atualizam em intervalos de tempo padrão, também conseguimos criar jogos com gráficos 2D e 3D animados, aplicativos, entre outros. Para fazer uma página web interativa, utilizamos HTML para fazer o conteúdo, CSS para fazer o estilo da página, e JavaScript para fazer a interação do cliente com o site.

O que é ECMAScript?

ECMAScript é uma especificação de linguagem de programação que define a estrutura, sintaxe e recursos JavaScript, é usada para criar scriptss que executam no cliente e no servidor, é usada para criar páginas web interativas, e também é usada para aplicativos e serviços do lado do servidor.
