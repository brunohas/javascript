O que é JavaScript?

JavaScript é uma linguagem de programação que permite a criação de conteúdos dinâmicos, controle de mídias e animações para deixar seu site mais interativo e interessante. Ele é uma das principais tecnologias da web, juntamente com o HTML e CSS.

O que faz JavaScript?

A linguagem de programação JavaScript permite ao desenvolvedor implementar diversos itens de alto nível de complexidade em páginas web como animações, mapas, menu suspenso, formulários, gráficos animados, informações que se atualizam em intervalos de tempo padrão, também conseguimos criar jogos com gráficos 2D e 3D animados, aplicativos, entre outros. Para fazer uma página web interativa, utilizamos HTML para fazer o conteúdo, CSS para fazer o estilo da página, e JavaScript para fazer a interação do cliente com o site.

O que é ECMAScript?

ECMAScript é uma especificação de linguagem de programação que define a estrutura, sintaxe e recursos JavaScript, é usada para criar scripts que executam no cliente e no servidor, é usada para criar páginas web interativas, e também é usada para aplicativos e serviços do lado do servidor.

O que é Node.js

Node.js é uma plataforma de desenvolvimento que permite a execução de código JavaScript no lado do servidor, ou seja, fora do navegador. Ele utiliza o motor V8 do Google Chorme para compilar e executar JavaScript, proporcionando uma maneira eficiente e escalável de construir aplicações web. A principal característica do Node.js é seu modelo de I/O (input/output) não bloqueante, que permite que múltiplas operações sejam realizadas simultaneamente, melhorando a performance e a capacidade de resposta das aplicações. Ele é utilizado para desenvolver aplicações web em tempo real, como chats de jogos online, onde a interação em tempo real é crucial. Também é ideal para a construção de APIs (Application Programming Interface), permitindo que diferentes sistemas se comuniquem de forma eficiente.

O que é o DOM?

O DOM (Document Object Model), basicamente é um conjunto de objetos dentro do seu navegador que vai dar acesso aos seus componentes internos do seu web site. O DOM é gerado pelo navegador quando carrega uma página, ele representa a página como uma árvore de nós, onde cada nó representa uma parte do documento. Permite alterar a estrutura, estilo e conteúdo do documento, também permite adicionar manipuladores de eventos aos nós, que são executados quando um evento ocorre.
